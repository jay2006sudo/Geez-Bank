## 📥 Installation Steps

### 1️⃣ Download the APK
- Click the download link above
- Wait for download to complete (~50-60 MB)
- The file saves to your Downloads folder

### 2️⃣ Allow Unknown Sources (First Time Only)

**Android 8.0 and above:**
- Settings → Apps & notifications → Advanced → Special app access → Install unknown apps
- Select your browser/file manager
- Toggle "Allow from this source" ON

**Android 7.x:**
- Settings → Security → Unknown Sources (toggle ON)

### 3️⃣ Install the App
- Open your Files app
- Navigate to Downloads
- Tap on `geez-bank-v1.0.0.apk`
- Tap "Install"
- Wait for installation (30-60 seconds)
- Tap "Done" or "Open"

### 4️⃣ Launch the App
- Find "Geez Bank" in your apps
- Tap to open
- Allow any permission requests
- Start using!

## 🔧 Troubleshooting

### Download Won't Start
- Check internet connection
- Try a different browser
- Clear browser cache
- Try downloading again

### Installation Fails with "Parse Error"
- Delete the downloaded file
- Download fresh copy
- Check available storage (150+ MB needed)

### "App Not Installed" Error
- Uninstall any previous versions first
- Restart phone and try again
- Check that file is complete (50+ MB)

### Won't Open After Install
- Force stop: Settings > Apps > Geez Bank > Force Stop
- Clear data: Settings > Apps > Geez Bank > Storage > Clear Data
- Uninstall and reinstall
- Restart device

### Permission Denied Errors
- Go to Settings > Apps > Geez Bank > Permissions
- Enable all requested permissions
- Restart the app
