# Geez Bank - Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Download
📱 **For Android Users:**
- Visit: https://github.com/jay2006sudo/Geez-Bank/releases
- Download the latest `.apk` file
- Tap to install on your phone

### Step 2: Create Account
1. Open the Geez Bank app
2. Tap "Sign Up"
3. Enter your details:
   - First Name
   - Last Name
   - Email
   - Password (8+ characters)
4. Tap "Create Account"

### Step 3: Start Banking
- ✅ View account balance
- ✅ Check transaction history
- ✅ Send money to others
- ✅ Manage multiple accounts

## 🏃 Common Tasks

### Check Your Balance
1. Open app and login
2. View balance on home screen
3. Tap on account for details

### Send Money
1. Tap "Transfer" button
2. Enter recipient details
3. Enter amount
4. Review and confirm
5. Done! Money sent.

### View Transactions
1. Tap "Transactions" tab
2. Scroll to see history
3. Pull down to refresh

### Update Profile
1. Tap "Profile" tab
2. View your information
3. Edit settings as needed
4. Logout when done

## ❓ FAQ

**Q: Is my data safe?**
A: Yes! All data is encrypted locally and transmitted via HTTPS.

**Q: Can I use on multiple devices?**
A: Yes, login with your email on any device.

**Q: What if I forget my password?**
A: Use the "Forgot Password" option on login screen.

**Q: How do I contact support?**
A: Open an issue on GitHub or email support@geezbank.com

**Q: Is there a web version?**
A: Coming soon! Stay tuned.
